# Quickstart Guide

This guide helps new users and developers get started quickly.

## Prerequisites

- Python >= 3.13
- Conda (optional)
- Required hardware (list specifics if any)

## Setup

1. **Clone the repository:**
   ```
   git clone https://github.com/malithjkd/pba_vision_mapping.git
   cd pba_vision_mapping
   ```

2. **Install dependencies:**
   - Using pip:
     ```
     python -m venv venv
     source venv/bin/activate  # or .\venv\Scripts\activate on Windows
     pip install -r requirements.txt
     ```
   - Or using Conda:
     ```
     conda env create -f environment.yml
     conda activate <env_name>
     ```

3. **Run the application:**
   ```
   python src/main.py
   ```

4. **Run tests:**
   ```
   pytest tests/
   ```
