# Test Plan: PBA Vision Mapping

This document outlines the formal test cases for validating the performance, accuracy, and repeatability of the PBA Vision Mapping system. Each test is designed to isolate and quantify a specific aspect of the system's performance.

| Test ID | Test Name | Description |
| :--- | :--- | :--- |
| **TST-001** | In-Position Vision Capability Analysis | **Objective:** Quantify the measurement repeatability of the vision system itself, independent of mechanical movement. <br> **Method:** At a single, stationary location, capture a series of images (e.g., 33 images) and calculate the standard deviation of the *dot radius*. <br> **Acceptance Criteria:** The 3-sigma value of the radius measurement is documented and meets the required precision threshold. |
| **TST-002** | In-Position Mechanical Jitter Analysis | **Objective:** Quantify the mechanical stability (vibration) of the system when holding a position. <br> **Method:** At a single, stationary location, capture a series of images (e.g., 33 images) and calculate the standard deviation of the *dot's X and Y center position*. <br> **Acceptance Criteria:** The 3-sigma value of the X and Y position variation is documented and falls within acceptable machine stability limits. |
| **TST-003** | Accurate Pixel-to-mm Calibration | **Objective:** Establish a highly accurate and stable `pixel-to-mm` conversion factor. <br> **Method:** In the cleanroom, scan a line of dots (e.g., 150 dots) using your "image stitching" method (`scan_method_5` + post-processing). Measure the physical distance between the first and last dot with the laser interferometer. Repeat multiple times to ensure consistency. <br> **Acceptance Criteria:** A stable `pixel-to-mm` value is determined with a variation of less than 0.1% across multiple tests. The methodology and final value are documented. |
| **TST-004** | Glass Certificate Generation Verification | **Objective:** Verify the stability and repeatability of the entire measurement system by generating multiple glass certificates. <br> **Method:** Using the newly calibrated `pixel-to-mm` ratio in the cleanroom, run the full glass certificate generation process at least 3 times. <br> **Acceptance Criteria:** The variation between the generated certificates is analyzed and documented. The results should be highly consistent, proving the end-to-end process is reliable. |
